🔐 AWS Serverless Contact Form API

Problem: Needed a scalable and secure way to handle contact form submissions from a static website and deliver messages via email without managing servers.

Solution: Built a serverless architecture using Amazon API Gateway and AWS Lambda (Python) to process form submissions, integrated with a static frontend hosted on Amazon S3 and delivered via Amazon CloudFront. Configured CORS for browser-based requests, implemented input validation, and integrated Amazon SES with a custom domain to send transactional emails.

Result: Delivered a fully serverless, production-style contact system capable of real-time email notifications using a custom domain, with no infrastructure to manage and aligned with AWS best practices for scalability, security, and reliability.
